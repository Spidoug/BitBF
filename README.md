# BitBF
BitBF - Bitcoin Wallet Key Scanner &amp; Validator 1.0

<img width="1037" height="843" alt="Captura de tela 2026-01-09 163513" src="https://github.com/user-attachments/assets/1dcdaad3-fd86-4a3d-bc0d-b003487aa141" />

Bitcoin Brute-Force and Validation tool written for the Processing (Java) environment. It combines a graphical user interface (GUI) with a multi-threaded engine to search for a specific Bitcoin address by iterating through private keys.
